Here's the (Espruino) pinouts I know:

| Pin No.  | Description |
| ------------- | ------------- |
|D2| analog battery (hi = 0.???, lo = ???)|
|D7 | SPI Flash MISO |
|D8 | SPI Flash MOSI |
|D9 | SPI Flash CLK |
|D16| backlight (1 == on, accepts analogWrite)|
|D22| battery on charger (0 = on charger) |
|D25| buzzer (1 == on, 0 == off, accepts analogWrite)|
|D29| BTN1 (1 == pressed)|
|D30| BTN2 (0 == pressed)|
